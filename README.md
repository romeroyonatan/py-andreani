## Desarrollo

1. git@github.com:romeroyonatan/py-andreani.git
2. mkdir ~/venv/
3. virtualenv -p python3 ~/venv/py-andreani/
4. source ~/venv/py-andreani/bin/activate
5. python setup.py install
